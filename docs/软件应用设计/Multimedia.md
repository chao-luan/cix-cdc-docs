# Multimedia

Multimedia 是 CIX P1 平台多媒体软件开发相关资源入口，主要面向视频编解码、图像处理、Camera、Display、Audio、流媒体处理和多媒体应用开发等场景。

CIX P1 Multimedia 相关能力基于 FFmpeg、GStreamer、NNStreamer 等多媒体框架，支持视频编解码和图像处理硬件加速，并支持 V4L2、VAAPI 以及 CIX Media Engine 等接口。

# PPA

PPA 是 CIX P1 平台软件包和驱动部署相关资源入口，用于在 Debian 系统上快速安装和更新 CIX 平台相关 Kernel、驱动、固件和硬件加速组件。

CIX PPA 支持通过软件源方式部署 CIX P1 平台相关软件包，减少手动编译、全镜像刷写和重复环境配置成本，适合开发者进行系统验证、驱动测试和开源软件栈体验。

# AI Model Hub

AI Model Hub 是面向 CIX P1 平台的 AI 模型资源入口，用于提供经过适配和优化的端侧 AI 模型，帮助开发者快速完成模型获取、部署验证和应用开发。

AI Model Hub 覆盖视觉、语音、LLM、多模态等典型端侧 AI 场景，适用于模型推理验证、Demo 开发、性能评估和应用集成。

## 官方入口

- [此芯开发者中心](https://developer.cixtech.com/)
- [CIX AI Model Hub](https://www.modelscope.cn/models/cix/ai_model_hub/files?version=26_Q1)

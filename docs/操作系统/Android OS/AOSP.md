# AOSP

AOSP 是 Android Open Source Project 的简称，是 Android 系统的开源基础版本。CIX P1 平台提供 Android OS 相关支持，可用于 Android 系统适配、硬件加速验证、外设驱动调试以及端侧 AI / 多媒体应用开发。

CIX P1 当前提供 `android-16.0.0_r2` 和 `kernel 6.6` 内核支持，并使能 VPU、DPU、GPU、NPU 等硬件加速能力，同时集成 BT / Wi-Fi、Camera、Audio、Display 等外设驱动支持。

# Panthor & Panfrost

Panthor & Panfrost 是 CIX P1 平台图形软件栈相关的开源支持内容。

Panthor 是面向 Arm Mali CSF GPU 的开源 Kernel DRM 驱动实现，Panfrost 是面向 Arm Mali GPU 的开源 OpenGL ES 驱动实现。CIX P1 支持 Arm Mali Avalon 架构 GPU，并支持 Panthor 和 Panfrost 开源图形栈。

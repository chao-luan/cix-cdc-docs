# Linux Kernel

Linux Kernel 是 CIX P1 平台操作系统适配的核心部分，负责硬件资源管理、设备驱动加载、进程调度、内存管理、文件系统以及系统调用等基础能力。

CIX P1 平台当前支持 Linux Kernel v6.1 和 v6.6，并支持 Device Tree 或 ACPI 等硬件描述方式。相关代码和开发分支可通过此芯开发者中心官网入口查看。

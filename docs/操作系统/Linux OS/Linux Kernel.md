# Linux Kernel

Linux Kernel 是 CIX P1 平台操作系统适配的核心组件，负责硬件资源管理、设备驱动加载、内存管理、进程调度、文件系统以及系统调用等基础能力。

CIX P1 平台提供 Linux Kernel 相关的开发资料、源码分支、patchset、版本说明和社区主线化支持入口。开发者可通过此芯开发者中心查看 OS 文档资源，也可通过开源支持页面查看 Linux Kernel 相关代码链接和开发计划。

## 官方入口

- [此芯开发者中心](https://developer.cixtech.com/)

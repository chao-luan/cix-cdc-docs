# 官网平台

瑞莎 Orion O6 是搭载 CIX P1 SoC 的开发平台，官网定位为开源 Arm V9 主板，可用于 CIX P1 平台的软件适配、系统验证、AI PC Demo、边缘 AI 应用开发以及开发者生态验证。

## 官网链接

- [Radxa Orion O6 官网平台](https://radxa.com/products/orion/o6/)

## 平台说明

该平台基于 CIX P1 SoC，采用 Armv9.2 架构，具备 12 核 CPU、Arm Immortalis G720 GPU、最高 45 TOPS 异构 AI 算力，并提供丰富的显示、网络、PCIe、USB 等接口资源，适合作为 CIX P1 开发者平台使用。

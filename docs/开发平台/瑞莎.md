# 官网平台

瑞莎 Orion O6 是搭载 CIX P1 平台的开发板，可作为 CIX P1 生态中的 SBC / Mini-ITX 开发平台使用。

本页面用于提供 Radxa Orion O6 官方产品资料入口，开发者可通过官网页面获取产品概览、AI PC 套件、技术规格、开发文档、下载资源、技术支持及配件信息等资料。

## 官网入口

- [Radxa Orion O6 官方产品页面](https://radxa.com/products/orion/o6/)

## 页面资料

官网页面中包含以下资料入口：

- Overview
- AI PC Kits
- Tech Spec
- Documentation
- Downloads
- Support
- Accessories

## 使用说明

进入官网入口后，可根据页面导航查看或下载 Orion O6 相关资料。

开发者可优先查看：

1. **Overview**：了解 Orion O6 产品定位与基础介绍。
2. **Tech Spec**：查看硬件规格、接口资源和平台能力。
3. **Documentation**：查看官方开发文档和使用说明。
4. **Downloads**：获取系统镜像、固件、工具及其他下载资源。
5. **Support**：查看官方支持信息和常见问题。
6. **Accessories**：查看适配配件和扩展资源。

# 官网平台

美高铭凡 MS-R1 是搭载 CIX P1 平台的开发设备，可作为 CIX P1 生态中的 Mini PC / Arm 工作站形态开发平台使用。

本页面用于提供 MS-R1 官方产品资料入口，开发者可通过官网页面获取规格、用户手册、驱动程序、下载资源、常见问题解答及教程视频等资料。

## 官网入口

- [MINISFORUM MS-R1 官方资料页面](https://www.minisforum.com/zh/pages/product-info)

## 页面资料

官网页面中包含以下资料入口：

- 规格
- 用户手册
- 驱动程序与下载
- 常见问题解答
- 教程视频

## 使用说明

进入官网入口后，在产品列表中选择 **MS-R1**，即可查看或下载对应资料。

开发者可优先查看：

1. **规格**：了解 MS-R1 的基础硬件配置与接口信息。
2. **用户手册**：查看设备使用、安装和维护说明。
3. **驱动程序与下载**：获取系统、驱动、BIOS、固件等相关资源。
4. **常见问题解答**：排查设备使用过程中的常见问题。
5. **教程视频**：参考官方操作演示完成设备配置。

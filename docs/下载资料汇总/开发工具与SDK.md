# 开发工具与SDK

本页面用于汇总 CIX P1 平台相关开发工具和 SDK 资源入口，包括 HIFI 5 DSP SDK、Multimedia SDK、GO SDK、NeuralOne AI SDK 和 Robotics SDK 等。

开发者可通过此芯开发者中心获取对应 SDK 的版本下载、开发文档、示例工程和使用说明，并根据实际业务场景选择合适的开发工具。

## 官方入口

- [此芯开发者中心](https://developer.cixtech.com/)

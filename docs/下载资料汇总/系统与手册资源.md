# 系统与手册资源

本页面用于汇总 CIX P1 平台相关的系统、固件、芯片手册、AI、图形、多媒体等开发与使用资料入口。

开发者可通过此芯开发者中心获取官方发布的文档资源，也可根据具体开发平台跳转至对应厂商官网查看系统镜像、BIOS、固件、驱动、手册和常见问题等资料。

## 官方入口

- [此芯开发者中心](https://developer.cixtech.com/)
